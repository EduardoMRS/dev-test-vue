# MCP (Model Context Protocol) de um Projeto Laravel + VUE

Este diretório contém a documentação estruturada de projeto Laravel + VUE, baseada no protocolo MCP, para facilitar o entendimento por IAs, automações e desenvolvedores.

## Conteúdo
- Visão geral do projeto
- Estrutura de pastas
- Rotas (backend e frontend)
- Controllers e Models
- Middlewares
- Estrutura de páginas Vue
- Exemplos de uso

## Atalhos para uso do docker
### igual a "php artisan" dentro do container
```bash
./art
```
### igual a "composer" dentro do container
```bash
./art pcomposer
```
### igual a "npm" dentro do container
```bash
./art npm
```
### igual a "mysql" loga e acessa dentro do container
```bash
./art mysql
```
### igual a "redis-cli" dentro do container
```bash
./art redis
```


## Service Worker Gerado Dinamicamente atraves de
```bash
./art service-worker:update
```
